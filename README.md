## 📦 Scripts

| 명령어               | 설명                 |
| :------------------- | :------------------- |
| `npm run dev`        | 개발 서버 실행       |
| `npm run build`      | 프로덕션 빌드        |
| `npm run preview`    | 빌드 결과물 미리보기 |
| `npm run lint`       | ESLint 실행          |
| `npm run test`       | Vitest 테스트 실행   |
| `npm run test:watch` | Vitest watch 모드    |
| `npm run test:ui`    | Vitest UI 모드       |
